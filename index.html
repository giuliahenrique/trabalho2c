C#

Logo:

Descrição:

Criada pela Microsoft em 2000, o C# (C Sharp) é uma linguagem moderna e orientada a objetos usada principalmente na plataforma .NET.

O que pode ser feito:

java

Logo:

Descrição:

C++

Descrição:

Desenvolvida por Bjarne Stroustrup em 1983, é uma evolução da linguagem C, com suporte a programação orientada a objetos.

O que pode ser feito:

Jogos, sistemas embarcados, softwares de alto desempenho, sistemas operacionais.

Assembly

Descrição:

Criada na década de 1940, o Assembly é uma linguagem de baixo nível que se comunica diretamente com o hardware. Cada instrução corresponde a uma operação do processador.

O que pode ser feito:

Programação de firmware, sistemas embarcados e drivers.
